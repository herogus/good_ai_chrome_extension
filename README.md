# 顾得助手 - AI 对话、网页总结、翻译与自动化工具

顾得助手是一款轻量级 Chrome 智能助手插件，支持多平台大语言模型接入，可在网页中进行 AI 对话、网页总结、全文翻译、考试辅助、网络调试和页面自动化等操作，帮助你更高效地阅读、理解和处理网页内容。

---

## 核心功能

### 多平台 AI 对话

支持配置多个主流大模型平台，在浏览网页时随时唤起 AI 助手进行对话。

支持的平台包括：

- OpenAI / ChatGPT
- Claude
- Gemini
- Ollama
- DeepSeek
- 阿里云通义千问
- 腾讯混元
- 智谱 GLM
- 字节豆包
- 百度文心一言
- OpenAI 兼容接口
- 自定义模型服务

支持流式响应、历史会话、模型切换、对话设置和小模型摘要等能力。

---

### 网页总结与页面理解

可自动提取当前网页内容，快速生成摘要，也可以结合你的问题进行定向分析。

适合：

- 阅读技术文档
- 总结长文章
- 分析博客、新闻、论文内容
- 快速理解网页核心信息

---

### 网页翻译助手

支持对当前页面进行翻译，并提供更细粒度的翻译范围控制。

主要能力：

- 一键翻译当前网页
- 自动识别源语言
- 支持选择目标语言
- 支持显示或隐藏原文
- 支持站点级自动翻译规则
- 支持翻译失败项重试
- 支持翻译范围模板

翻译范围可按页面区域控制，例如正文、导航、页眉、页脚、目录、表格、评论、代码块等。你可以使用系统内置模板，也可以创建自定义模板。

---

### 智能悬浮窗与侧边栏

插件提供两种主要使用形态：

- 页面悬浮窗：轻量显示，不打断网页浏览
- 浏览器侧边栏：更大的操作空间，适合长对话和深度阅读

支持：

- 悬浮窗拖动
- 全屏展开
- 极简模式
- 当前页面关闭悬浮窗
- 全局关闭悬浮窗
- 侧边栏快速展开
- 历史会话查看与切换

---

### 考试辅助模式

针对在线考试、练习题和页面题目场景，可结合页面内容分析题目，给出参考答案和解题思路。

适合：

- 在线学习
- 题目分析
- 考试辅助
- 练习复盘

---

### 智能网络调试助手

面向开发者的页面网络调试能力，可监听页面请求并辅助分析接口数据。

支持：

- 查看页面请求日志
- 添加请求拦截规则
- 配置请求头、请求体、响应体
- 配置重定向地址
- 辅助 Mock 接口数据

适合前端开发、接口联调和页面调试。

---

### 页面自动化助手

支持配置页面自动化流程，用于记录和执行常见页面操作。

可用于：

- 页面点击
- 输入内容
- 等待
- 滚动
- 提取数据
- 录制页面操作流程

适合重复性网页操作和简单自动化任务。

---

### 模型配置管理

提供独立的模型管理界面，方便维护不同模型配置。

支持：

- 新增模型配置
- 编辑模型配置
- 删除模型配置
- 导入 / 导出模型配置
- 标记小模型
- 切换当前使用模型

---

### 主题与外观设置

提供完整的外观配置能力。

支持：

- 跟随系统主题
- 浅色模式
- 深色模式
- 多套主题色
- 自定义对话背景图片
- 暗黑模式下单独控制是否显示背景
- 背景遮罩强度调节

---

### 常用工具

设置中心内置常用小工具，方便日常处理文本内容。

当前支持：

- JSON 格式化
- JSON 压缩
- URL 编码 / 解码
- Base64 编码 / 解码

---

### 会话历史管理

自动保存聊天历史，方便回顾和继续之前的对话。

支持：

- 按日期分组
- 编辑会话标题
- 删除会话
- 导出会话
- 清空历史记录
- 快速切换历史会话

## 权限说明

为保证功能正常运行，插件需要申请以下浏览器权限：

| 权限名称 | 用途说明 |
| --- | --- |
| `activeTab` | 获取当前激活标签页的信息，用于网页总结、翻译、考试辅助等主动触发的功能。 |
| `tabs` | 识别当前标签页状态，打开侧边栏、设置页或辅助页面。 |
| `storage` | 本地保存模型配置、主题设置、翻译规则、聊天历史等数据。 |
| `sidePanel` | 在 Chrome 侧边栏中展示插件主界面。 |
| `scripting` | 在当前页面注入内容脚本，用于悬浮窗、翻译、页面内容读取等能力。 |
| `unlimitedStorage` | 支持保存较大的本地配置，例如自定义背景图片。 |

> 隐私说明：插件不会自动上传网页内容。网页总结、翻译、考试辅助等操作均由用户主动触发。API Key、模型配置、聊天历史等数据默认保存在本地浏览器中，不会被插件主动上传或共享。

## 使用场景

- 学习：总结技术文档、翻译外文资料、辅助理解复杂内容

- 翻译：网页全文翻译、站点自动翻译、翻译范围精细控制
- 自动化：记录并执行重复性的网页操作
- 内容创作：结合网页内容进行整理、提炼和改写
